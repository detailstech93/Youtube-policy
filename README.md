# Details Tech Repository

This repository contains the public review and sponsored content policy for **Details Tech** in both Arabic and English.

## Files
- `ar/policy-ar.md` — Arabic version
- `en/policy-en.md` — English version

## Suggested GitHub setup
1. Create a new repository on GitHub.
2. Upload these files.
3. Add a short description such as: `Public review and sponsorship policy for Details Tech`.
4. Set the repository to Public if you want this policy visible to everyone.
5. Add a `README.md` if you want a landing page in GitHub.

## Optional next steps
- Add a `LICENSE` file if you want to define reuse rights.
- Add a `CONTRIBUTING.md` if you expect collaborators.
- Turn on GitHub Pages if you want a simple public webpage.
